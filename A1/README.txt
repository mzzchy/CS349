To compile the game, run command line "make" or "make run"
To run the game, either click the icon or run command line "./game"
To clean additional file, run command line "make clean"

The tile is generated randomly.Once the player passed 60 tiles, the game will end
Every time the chopper hits an enemy, the player gain 10 score point.

Instruction:
Press "space" to drop a bomb, a player can only drop 6 bombs at a time
Press G for immunity/God mode.
Press S to start
Press F to show help and student id.

Additional Feature: Score board, win/lose text.
